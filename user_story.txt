1. Пользовательская история для регистрации клиента
Как новый пользователь,
Я хочу зарегистрироваться на сайте массажного салона,
Чтобы иметь возможность бронировать комнаты для массажа и управлять своими записями.

Критерии приемки:

Поля регистрации включают: имя, номер телефона, email и пароль.
После успешной регистрации данные сохраняются в системе, и пользователь видит сообщение об успешной регистрации.
Если email уже зарегистрирован, отображается ошибка.

2. Пользовательская история для бронирования комнаты
Как зарегистрированный пользователь,
Я хочу забронировать комнату для массажа на конкретную дату и время,
Чтобы быть уверенным, что комната будет свободна для меня.

Критерии приемки:

Пользователь может выбрать только те комнаты, которые свободны на выбранную дату и время.
После бронирования статус комнаты меняется на "В брони".
Пользователь получает подтверждение о бронировании с деталями: номер комнаты, дата и время.
Если пользователь пытается забронировать занятую комнату, отображается сообщение об ошибке.

3. Пользовательская история для просмотра списка услуг
Как пользователь,
Я хочу просматривать список услуг массажного салона с описанием и ценами,
Чтобы выбрать подходящий тип массажа перед бронированием.

Критерии приемки:

Список услуг отображает название, краткое описание и цену каждой услуги.
Пользователь может фильтровать услуги по типу массажа (например, обычный или VIP).
Список доступен без необходимости регистрации.

4. Пользовательская история для управления бронированием
Как зарегистрированный пользователь,
Я хочу видеть свои текущие брони и иметь возможность отменить их,
Чтобы изменить свои планы при необходимости.

Критерии приемки:

Пользователь видит список своих текущих броней с информацией о дате, времени и номере комнаты.
Пользователь может выбрать бронь и отменить её.
После отмены статус комнаты автоматически меняется на "Свободно".
Пользователь получает подтверждение об успешной отмене.

5. Пользовательская история для просмотра массажисток
Как пользователь,
Я хочу видеть информацию о массажистках, которые работают в салоне,
Чтобы выбрать подходящего специалиста перед бронированием.

Критерии приемки:

Пользователь видит список массажисток с их именами и привязкой к комнатам.
Для каждой массажистки отображается базовая информация: имя и опыт работы (если добавлено).
Список обновляется автоматически при изменении прикрепления массажисток к комнатам.
